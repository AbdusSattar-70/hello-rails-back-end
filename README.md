# 📖 Hello-rails-back-end<a name="about-project"></a>

The provided API endpoint serves as the setup for the "hello-react-front-end" project, which is designed for educational and learning purposes. As an API endpoint, it acts as a bridge between the front-end application (hello-react-front-end) and the backend server, allowing them to communicate and exchange data.

API endpoints are essential components of modern web applications as they facilitate data retrieval, storage, and manipulation. In this specific case, the endpoint is responsible for handling requests from the front-end application and providing appropriate responses.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Mac or PC
- Code Editor (ex: Vs Code)
- Terminal
- Postgresql
- Node.js
- git
- Install [ruby](https://www.ruby-lang.org/en/documentation/installation/) on your computer
- Install [rails](https://www.tutorialspoint.com/ruby-on-rails/rails-installation.htm#) on your computer

### Setup

Clone this repository to your desired folder:

```sh
  cd your-desired-folder
  https://github.com/AbdusSattar-70/hello-rails-back-end.git
```

### Install

- Install this project with:

- Linters
- Rubocop
- Ruby On Rails

- Follow the step:

```sh
cd hello-rails-back-end
npm install
bundle install
rails s
```

### Usage

- Run the app with `rails s` in the terminal from the root folder of the project.

### Test

```sh
  rspec spec/filename.rb
```
